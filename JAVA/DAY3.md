*배열*

> 순서대로, 특정상자에 접근하기 위해



{0, 1, 2, 3, 4} > 0부터 시작하며, Index 라고함 



* 공간을 저장하는 방법 3가지 

```
int [] box = {10,20};
int [] box = new int[] {10,20};
int [] box = new int[2];
box[0] = 10;
box[1] = 20
```





* 읽는 법 

```
int [] box = new int [3];

box[0] = 10;
box[1] = 20,
box[2] = 30; 
```
