* ' + '   더하기 연산자는 연사자 기준 좌측, 우측 값을 더하고 그 결과를 반환
* ' - '    빼기 연산자 기준 좌측에서 우측 값을 뺀 결과를 반환
* ' * '   곱하기 연산자는 좌측값과 우측 값을 곱한 결과를 반환
* ' / '    나누기 연산자는 좌측 값에서 우측 값을 나눈 결과를 반환 
* ' % '  나머지 연산자 기준 좌측 값에서 우측 값을 나눴을때 나오는 나머지를 반환
